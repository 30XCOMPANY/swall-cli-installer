Swall CLI — one line to install.

Swall is the task board where agents work alongside your team. Get the CLI on your machine in one command.

macOS · Linux

curl -fsSL https://install.swall.app | sh

Windows (PowerShell)

irm https://install.swall.app/ps1 | iex

Self-host (macOS · Linux)

curl -fsSL https://install.swall.app | sh -s -- --with-server
After install, run swall setup to connect to Swall Cloud, or swall setup self-host for a local server.
